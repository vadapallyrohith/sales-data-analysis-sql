# 📊 Sales Data Analysis using SQL

## 📌 Objective
Analyze retail sales data to identify trends and key business insights.

## 🛠 Tools Used
- SQL (MySQL)

## 📊 Analysis Performed
- Total and average sales
- Sales by region
- Sales by state
- Top-performing cities
- Profit analysis

## 🔍 Key Insights
- West region generates the highest sales
- Sales are concentrated in top cities
- Certain categories drive more profit

## 📁 Files
- SQL queries file
- Dataset

## 🚀 Outcome
Demonstrates ability to analyze structured data and extract meaningful insights using SQL.
