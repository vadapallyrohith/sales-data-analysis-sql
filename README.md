# 📊 Sales Data Analysis using SQL

## 📌 Objective
Analyze retail sales data to identify trends and key business insights.

## 🛠 Tools Used
- SQL (MySQL)

## 📊 Analysis Performed
- Total and average sales
- Sales by region
- Sales by state
- Top-performing cities
- Profit analysis

## 🔍 Key Insights
- West region generates the highest total sales among all regions
- Sales are concentrated in a few top-performing cities
- Technology category contributes higher profit compared to others
- A small number of locations drive a large portion of revenue

📊 Regional Insight
“Sales are highest in the West region, indicating strong market performance compared to other regions.”
🏙️ Geographic Insight
“Revenue is concentrated in a few top-performing cities, suggesting dependency on specific markets.”
📦 Category Insight
“Certain categories generate high sales but relatively lower profit, indicating margin inefficiencies.”
📉 Profitability Insight
“Some regions and categories show consistently low profit, highlighting areas for cost optimization.”
📊 Distribution Insight
“Significant variation between minimum and maximum sales across regions indicates uneven demand distribution.”
